# RespectNOW

un simplu launcher de jocuri si aplicatii facut de mine si de sorin.schema, dar mai este de facut la el

ca sa porniti acest launcher va trebui sa instalati cateva chestii in python mai jos o sa va las tot ce trebuie instalat

1. python -m pip install customtkinter
2. python -m pip install pillow
3. python -m pip install requests
4. atat trebuie sa instalati, ca sa porniti launcher ul e simplu deschideti cmd u din proiect si da ti comanda asta python main.pyw, nu este cine stie ce dar e ceva ok de inceput pentru cei care vor sa faca launcher
de jocuri si aplicatii
